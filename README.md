# packer-aws-runner

Basic CICD node for using Hashicorp's Packer as GitLab runner in AWS

## What's included

* [Alpine Linux](https://alpinelinux.org/) 3.10
* [HashiCorp Packer](https://packer.io/) 1.5.1
* [Goss](https://github.com/aelsabbahy/goss/) 0.3.9


## Additional tools

* curl
* jq
* rsync
